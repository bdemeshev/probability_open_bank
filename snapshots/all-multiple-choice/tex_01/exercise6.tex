
\begin{question}
Обозначим точное P-значение буквой \(V\).
Чему равна вероятность \(\mathbb{P} (V < 0.1 \mid H_0)\)?
\begin{answerlist}
  \item 0.3
  \item нет верного ответа
  \item 0.8
  \item 0.1
  \item 0.5
  \item 0.2
\end{answerlist}
\end{question}

\begin{solution}
При верной \(H_0\) P-значение имеет равномерное распределение.
\end{solution}
